Условные конструкции

Любая программа, от простого скрипта до сложного приложения, должна уметь принимать решения. "Если пользователь вошел в систему, показать его профиль, иначе показать форму входа". "Если результат вычислений отрицательный, выделить его красным цветом". Для реализации этой логики в JavaScript используются условные конструкции.

Булева логика: основа условий

Прежде чем изучать условные операторы, важно понять тип данных, который лежит в их основе — Boolean (логический тип). Он может принимать всего два значения: true (истина) и false (ложь).

Условие внутри оператора if  всегда вычисляется в одно из этих значений. Выражения, которые возвращают true или false - операторы сравнения:

  • > (больше), < (меньше)
  • >= (больше или равно), <= (меньше или равно)
  • == (равно по значению, не строгое)
  • === (равно по значению и типу, строгое равенство)
  • != (не равно по значению)
  • !== (не равно по значению и типу)

console.log(10 > 5); // true
console.log(10 === 10); // true
console.log(10 === '10'); // false (разныетипы)
console.log(10 != 20); // true

Прямое указание булева значения:

let isLoaded = true;
if (isLoaded) { ... } // условие выполнится

Конструкция if

Конструкция if — фундаментальный инструмент для ветвления логики программы. Она выполняет блок кода только если указанное условие истинно.

Синтаксис:

if (условие) {
// Тело оператора if
// Код, выполняемый, если условие истинно (true)
}
// Код, который выполнится в любом случае

Пример и пошаговое выполнение:

let currentTemperature = 25;

// 1. Вычисляетсяусловие: (currentTemperature > 20)
// 2. currentTemperature = 25, поэтому (25 > 20) = true
// 3. Поскольку условие истинно, код внутри блока выполняется

if (currentTemperature > 20) {
console.log("На улице тепло. Можно идти в футболке."); // Это сообщение будет выведено
}
console.log("Проверка погоды завершена."); // Этот код выполнится в любом случае

Важное замечание о блоках кода: Фигурные скобки {} определяют блок кода, который ассоциируется с оператором if. Если внутри блока всего одна инструкция, скобки можно опустить. Однако настоятельно рекомендуется всегда их использовать, особенно новичкам. Это делает код более читаемым, предотвращает ошибки и упрощает его дальнейшее расширение.

Конструкция if...else

Часто требуется предусмотреть не только то, что произойдет при истинности условия, но и то, что случится в противном случае. Для этого используется расширенная конструкция if...else ("если...иначе").

Синтаксис:

if (условие) {
// Блок if: выполняется, если условие истинно
} else {
// Блок else: выполняется, если условие ложно
}
// Код после конструкции

Пример:

let age = 16;
if (age >= 18) {
console.log("Вы совершеннолетний. Доступ разрешен.");
} else {
// Этот блок выполнится, так как (16 >= 18) = false
console.log("Вы несовершеннолетний. Доступ запрещен.");
}
console.log("Проверка возраста завершена.");

Последовательная проверка: else if

Когда существует не два, а три и более возможных сценария, на помощь приходит конструкция else if. Она позволяет проверить несколько условий подряд.

Синтаксис:

if (условие1) {
// Выполняется, если условие1 истинно
} else if (условие2) {
// Выполняется, если условие1 ложно, но условие2 истинно
} else if (условие3) {
// Выполняется, если условие1 и условие2 ложны, но условие3 истинно
} else {
// Выполняется, если все вышеперечисленные условия ложны
}

Пример:

let testScore = 92;
let grade;
if (testScore >= 90) {
grade = "A";
console.log("Отлично!");
} else if (testScore >= 80) { // Проверяется, только если score < 90
grade = "B";
console.log("Хорошо!");
} else if (testScore >= 70) { // Проверяется, только если score < 80
grade = "C";
console.log("Удовлетворительно.");
} else { // Сработает, только если все условия выше ложны (score < 70)
grade = "F";
console.log("Неудовлетворительно. Необходимо повторить материал.");
}
console.log(`Ваша оценка: ${grade}`);

Ключевой момент: Выполнение переходит только к одному блоку. Интерпретатор проверяет условия сверху вниз. Как только находится первое истинное условие, выполняется соответствующий ему блок, а все последующие else if и else игнорируются.

Вложенные условия

Условные операторы можно помещать внутрь других условных операторов. Это называется вложенностью.

Пример: Допустим, мы хотим дать скидку только зарегистрированным пользователям, если их корзина превышает определенную сумму.

let isUserLoggedIn = true;
let cartTotal = 1500;
if (isUserLoggedIn) {
// Этот блок выполняется, если пользователь вошел в систему
if (cartTotal > 1000) {
console.log("Вам предоставляется скидка 10%!");
} else {
console.log("Добавьте товаров на сумму более 1000 руб. для получения скидки.");
}
} else {
console.log("Для получения скидки необходимо авторизоваться.");
}

Вложенность может быть очень глубокой, но важно не злоупотреблять ею, так как это ухудшает читаемость кода.

Логические операторы для составных условий

Часто условие должно состоять из нескольких частей. Для их объединения используются логические операторы.

  • && (Логическое И): Возвращает true, только если оба операнда истинны.
let isWeekend = true;
let isSunny = false;
if (isWeekend && isSunny) {
console.log("Идеальный день для пикника!"); // Не выполнится (isSunny = false)
}
  • || (Логическое ИЛИ): Возвращает true, если хотя бы один операнд истинен.
let hasCreditCard = false;
let hasEnoughCash = true;
if (hasCreditCard || hasEnoughCash) {
console.log("Оплата прошла успешно."); // Выполнится (hasEnoughCash = true)
}
  • ! (Логическое НЕ): Инвертирует булево значение. true становится false, и наоборот.
let isUserBanned = false;
if (!isUserBanned) { // !false = true
console.log("Добро пожаловать на сайт!"); // Выполнится
}

Комбинирование операторов: С помощью круглых скобок можно создавать сложные комплексные условия, задавая приоритет их выполнения.

let age = 25;
let hasDriverLicense = true;
// "Если возраст больше 18 И (имеются права ИЛИ человек старше 65)"
if (age > 18 && (hasDriverLicense || age > 65)) {
console.log("Можно арендовать автомобиль.");
}

Конструкция множественного выбора switch

Конструкция switch предоставляет альтернативный способ организации ветвления программы, когда нужно сравнить одно значение с множеством возможных вариантов.

Базовый синтаксис

switch (выражение) {
case значение1:
// код, выполняемый если выражение === значение1
break;
case значение2:
// код, выполняемый если выражение === значение2
break;
default:
// код, выполняемый если ни один case не подошел
}

Как это работает:

  • Вычисляется выражение в скобках
  • Его значение последовательно сравнивается со значениями в блоках case (используется строгое сравнение ===)
  • Если совпадение найдено, выполняется код соответствующего блока case
  • Ключевое слово break прерывает выполнение конструкции switch
  • Если совпадений не найдено, выполняется код в блоке default (если он есть)

Пример использования

let dayOfWeek = 3;
let dayName;
switch (dayOfWeek) {
case 1:
dayName = "Понедельник";
break;
case 2:
dayName = "Вторник";
break;
case 3:
dayName = "Среда"; // Это значение будет выбрано
break;
case 4:
dayName = "Четверг";
break;
case 5:
dayName = "Пятница";
break;
case 6:
dayName = "Суббота";
break;
case 7:
dayName = "Воскресенье";
break;
default:
dayName = "Неизвестный день";
}
console.log(dayName); // "Среда"

Особенности конструкции switch

"Проваливание" (fallthrough)

Если не использовать break, выполнение продолжится в следующем блоке case:

let rating = 4;
let message = "";
switch (rating) {
case 5:
message += "Отлично! ";
case 4:
message += "Очень хорошо! ";
case 3:
message += "Хорошо! ";
case 2:
message += "Удовлетворительно. ";
case 1:
message += "Плохо. ";
break;
default:
message = "Неизвестная оценка";
}
console.log(message); // "Очень хорошо! Хорошо! Удовлетворительно. Плохо. "

Группировка case

Несколько условий case можно группировать для выполнения одного блока кода:

let fruit = "Яблоко";
let type;
switch (fruit) {
case "Апельсин":
case "Мандарин":
case "Лимон":
type = "Цитрусовые";
break;
case "Яблоко":
case "Груша":
type = "Семечковые"; // Это значение будет выбрано
break;
case "Вишня":
case "Черешня":
type = "Косточковые";
break;
default:
type = "Неизвестный тип";
}
console.log(type); // "Семечковые"

Когда использовать switch вместо if...else

Используйте switch, когда:

  • Есть одна переменная/выражение для сравнения
  • Много возможных значений для сравнения (более 3-4)
  • Нужна лучшая читаемость для множественных условий

Используйте if...else, когда:

  • Нужно проверить диапазоны значений
  • Условия сложные и включают логические операторы
  • Нужно проверить разные переменные в одном условии

Задания для самостоятельной работы:

1. Напишите программу, которая проверяет переменную num.
Если число положительное, выведите "Положительное".
Если число отрицательное, выведите "Отрицательное".
Если число равно нулю, выведите "Ноль". 

let num = 0; // Здесь можно изменить значение для проверки
if (num > 0) {
    console.log("Положительное");
} else if (num < 0) {
    console.log("Отрицательное");
} else {
    console.log("Ноль");
}

2. Пользователь вводит текущий час (число от 0 до 23). Напишите код, который выводит приветствие в зависимости от времени:
С 6 до 12 – "Доброе утро!"
С 13 до 17 – "Добрый день!"
С 18 до 22 – "Добрый вечер!"
В остальное время – "Доброй ночи!" 

// Получаем текущий час от пользователя
let hour = parseInt(prompt("Введите текущий час (число от 0 до 23):"));

// Проверяем корректность ввода
if (isNaN(hour) || hour < 0 || hour > 23) {
    console.log("Ошибка: введите число от 0 до 23");
} else {
    // Проверяем диапазоны времени и выводим приветствие
    if (hour >= 6 && hour <= 12) {
        console.log("Доброе утро!");
    } else if (hour >= 13 && hour <= 17) {
        console.log("Добрый день!");
    } else if (hour >= 18 && hour <= 22) {
        console.log("Добрый вечер!");
    } else {
        console.log("Доброй ночи!");
    }
}

3. Напишите программу, которая симулирует процесс входа.
1) Создайте константы CORRECT_LOGIN = "admin и CORRECT_PASSWORD = "qwerty123".
2) Запросите у пользователя логин и пароль через prompt.
3) Если логин и пароль совпадают с корректными, выведите "Доступ разрешен".
4) Если логин верный, но пароль нет – "Неверный пароль".
5) Если логин неверный – "Пользователь с таким логином не найден". Подсказка: Используйте вложенные условия. 

// Создаем константы с правильными логином и паролем
const CORRECT_LOGIN = "admin";
const CORRECT_PASSWORD = "qwerty123";

// Запрашиваем у пользователя логин и пароль
let userLogin = prompt("Введите логин:");
let userPassword = prompt("Введите пароль:");

// Проверяем введенные данные
if (userLogin === CORRECT_LOGIN) {
    // Если логин верный, проверяем пароль
    if (userPassword === CORRECT_PASSWORD) {
        console.log("Доступ разрешен");
    } else {
        console.log("Неверный пароль");
    }
} else {
    console.log("Пользователь с таким логином не найден");
}

4. Напишите программу, которая запрашивает у пользователя номер месяца (1-12) и использует switch для вывода названия времени года:
- 12, 1, 2: "Зима"
- 3, 4, 5: "Весна"
- 6, 7, 8: "Лето"
- 9, 10, 11: "Осень" 

let month = parseInt(prompt("Введите номер месяца (от 1 до 12):"));

if (isNaN(month) || month < 1 || month > 12) {
    console.log("Ошибка: введите число от 1 до 12");
} else {
    switch (month) {
        case 12: case 1: case 2:
            console.log("Зима");
            break;
        case 3: case 4: case 5:
            console.log("Весна");
            break;
        case 6: case 7: case 8:
            console.log("Лето");
            break;
        case 9: case 10: case 11:
            console.log("Осень");
            break;
    }
}

5. Создайте простой калькулятор, который:
1) Запрашивает первое число
2) Запрашивает оператор (+, -, *, /, %)
3) Запрашивает второе число
4) Использует switch для выполнения операции и вывода результата 

let num1 = parseFloat(prompt("Введите первое число:"));
let operator = prompt("Введите оператор (+, -, *, /, %):");
let num2 = parseFloat(prompt("Введите второе число:"));

if (isNaN(num1) || isNaN(num2)) {
    console.log("Ошибка: введите корректные числа");
} else {
    let result;
    let isValid = true;
    
    switch (operator) {
        case '+':
            result = num1 + num2;
            break;
        case '-':
            result = num1 - num2;
            break;
        case '*':
            result = num1 * num2;
            break;
        case '/':
            if (num2 === 0) {
                console.log("Ошибка: деление на ноль невозможно!");
                isValid = false;
            } else {
                result = num1 / num2;
            }
            break;
        case '%':
            if (num2 === 0) {
                console.log("Ошибка: деление на ноль при вычислении остатка!");
                isValid = false;
            } else {
                result = num1 % num2;
            }
            break;
        default:
            console.log("Ошибка: неверный оператор. Допустимые операторы: +, -, *, /, %");
            isValid = false;
    }
    
    if (isValid) {
        // Округляем результат для удобства чтения
        if (Number.isInteger(result)) {
            console.log(`${num1} ${operator} ${num2} = ${result}`);
        } else {
            console.log(`${num1} ${operator} ${num2} = ${result.toFixed(4)}`);
        }
    }
}
