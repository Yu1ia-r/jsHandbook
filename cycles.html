Циклы

Введение в циклы

Циклы — это фундаментальная концепция программирования, позволяющая выполнять один и тот же блок кода многократно. Они незаменимы при обработке данных, работе с коллекциями элементов и автоматизации повторяющихся задач. В JavaScript существует несколько видов циклов, но в этой главе мы сосредоточимся на двух основных: while и for.

Цикл while

Цикл while выполняет блок кода до тех пор, пока заданное условие истинно. Это универсальный цикл, который особенно полезен, когда количество итераций заранее неизвестно.

Базовый синтаксис

while (условие) {
// Тело цикла - код, который будет повторяться
}

Как работает цикл while

  • Проверяется условие в круглых скобках
  • Если условие истинно (true), выполняется код в теле цикла
  • После выполнения тела цикла условие проверяется снова
  • Процесс повторяется, пока условие не станет ложным (false)
  • Как только условие становится ложным, выполнение цикла прекращается

Примеры использования

Пример 1: Простой счетчик

let counter = 1;
while (counter <= 5) {
console.log("Итерация номер: " + counter);
counter++; // Увеличиваем counter на 1
}

Пример 2: Обработка пользовательского ввода

let userInput = "";
const secretPassword = "qwerty123";
while (userInput !== secretPassword) {
userInput = prompt("Введите пароль:");
}
console.log("Доступ разрешен!");

Пример 3: Обработка данных до достижения условия

let total = 0;
let number = 1;
while (total < 100) {
total += number;
number++;
console.log(`Текущая сумма: ${total}`);
}
console.log(`Для достижения суммы 100 потребовалось ${number-1} чисел`);

Важные особенности и предостережения

1. Бесконечные циклы: Если условие всегда остается истинным, цикл будет выполняться бесконечно, что может привести к зависанию программы.

// БЕСКОНЕЧНЫЙ ЦИКЛ - НЕ ВЫПОЛНЯТЬ!
while (true) {
console.log("Этот текст будет выводиться вечно");
}

2. Изменение условия: Важно обеспечить, чтобы в теле цикла было что-то, что в конечном итоге сделает условие ложным.

3. Проверка условия до выполнения: Цикл while проверяет условие перед каждой итерацией. Если условие изначально ложно, тело цикла не выполнится ни разу.

Цикл for

Цикл for предоставляет более компактный синтаксис для создания циклов и особенно полезен, когда количество итераций известно заранее или зависит от счетчика.

Базовый синтаксис

for (инициализация; условие; завершающее_выражение) {
// Тело цикла
}
  • Инициализация: Выполняется один раз перед началом цикла. Обычно здесь объявляется и инициализируется счетчик.
  • Условие: Проверяется перед каждой итерацией. Если истинно, выполняется тело цикла.
  • Завершающее выражение: Выполняется после каждой итерации. Обычно здесь изменяется значение счетчика.

Как работает цикл for

  • Выполняется выражение инициализации
  • Проверяется условие
  • Если условие истинно, выполняется тело цикла
  • Выполняется завершающее выражение
  • Возврат к шагу 2 (проверка условия)

Примеры использования

Пример 1: Простой счетчик

for (let i = 1; i <= 5; i++) {
console.log("Итерация номер: " + i);
}

Пример 2: Перебор элементов массива

const fruits = ["Яблоко", "Банан", "Апельсин", "Виноград"];
for (let i = 0; i < fruits.length; i++) {
console.log(`Фрукт ${i + 1}: ${fruits[i]}`);
}

Пример 3: Обратный отсчет

for (let countdown = 10; countdown > 0; countdown--) {
console.log(countdown);
}
console.log("Старт!");

Вариации цикла for

Несколько переменных в инициализации:

for (let i = 0, j = 10; i < j; i++, j--) {
console.log(`i = ${i}, j = ${j}`);
}

Пропуск частей цикла:

let i = 0;
for (; i < 5;) { // Можно пропустить инициализацию и завершающее выражение
console.log(i);
i++;
}

Бесконечный цикл for:

// БЕСКОНЕЧНЫЙ ЦИКЛ - НЕ ВЫПОЛНЯТЬ!
for (;;) {
console.log("Этот текст будет выводиться вечно");
}

Управление выполнением циклов: break и continue

Внутри циклов можно использовать два специальных оператора для управления выполнением:

Оператор break

Немедленно прерывает выполнение цикла и передает управление следующей за циклом инструкции.

Пример:

for (let i = 1; i <= 10; i++) {
if (i === 6) {
console.log("Цикл прерван на числе 6");
break;
}
console.log(i);
}
// Выведет: 1, 2, 3, 4, 5, "Цикл прерван на числе 6"

Оператор continue

Пропускает текущую итерацию цикла и переходит к следующей.

Пример:

for (let i = 1; i <= 5; i++) {
if (i === 3) {
continue; // Пропустить итерацию с i = 3
}
console.log(i);
}
// Выведет: 1, 2, 4, 5

Вложенные циклы

Циклы можно помещать внутрь других циклов, создавая вложенные конструкции. Это полезно для работы с многомерными структурами данных.

Пример: Таблица умножения

for (let i = 1; i <= 10; i++) {
let row = "";
for (let j = 1; j <= 10; j++) {
row += (i * j) + "\t"; // \t - символ табуляции
}
console.log(row);
}

Выбор между while и for

  • Используйте while, когда количество итераций неизвестно заранее и зависит от условия, которое может измениться в теле цикла.
  • Используйте for, когда количество итераций известно заранее или зависит от счетчика.
