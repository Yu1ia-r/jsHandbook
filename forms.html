Работа с формами

Формы являются ключевым элементом для сбора пользовательских данных, и JavaScript предоставляет мощные инструменты для работы с ними. Существуют специальные события, которые позволяют реагировать на действия пользователя в формах.

Главные события форм

  • submit: Это событие срабатывает при попытке отправки формы (например, при нажатии на <button type="submit">). Оно идеально подходит для финальной проверки данных. Для отмены стандартного поведения браузера (перезагрузки страницы) используется метод event.preventDefault().
  • change: Это событие срабатывает, когда значение элемента формы изменилось, и этот элемент потерял фокус. Оно очень полезно для валидации отдельных полей, как только пользователь закончил ввод.
  • focus и blur: Эти события срабатывают, когда элемент получает и теряет фокус соответственно. Они идеально подходят для предоставления мгновенной обратной связи, например, для изменения стиля поля или показа подсказки.

Опытные разработчики используют эти события в сочетании. Например, focus и blur для визуальных эффектов (подсветка поля), change для быстрой валидации отдельных полей, и submit для окончательной валидации всей формы.

Задания для самостоятельной работы:

 <form id="login-form">
    <input type="email" id="email" placeholder="Ваш email">
    <span id="email-hint" style="color: grey; font-size: 0.8em;"></span>
    <br>
    <input type="password" id="password" placeholder="Пароль">
    <br>
    <button type="submit">Войти</button>
</form>
<div id="message"></div>
1) На событие focus поля #email показывайте в #email-hint текст "Введите корректный email".
2) На событие blur поля #email убирайте этот текст.
3) На событие change поля #password проверяйте длину пароля. Если меньше 6 символов, подсвечивайте поле красной рамкой (el.style.border = '2px solid red'), иначе — убирайте подсветку.
4) На событие submit формы отменяйте стандартную отправку (preventDefault()), проверяйте, что email не пустой и пароль >= 6 символов. Если всё ок, выводите в #message текст "Форма успешно отправлена!" зеленым цветом. Если есть ошибки — выводите "Исправьте ошибки в форме" красным.

 const form = document.getElementById('login-form');
const emailInput = document.getElementById('email');
const emailHint = document.getElementById('email-hint');
const passwordInput = document.getElementById('password');
const messageDiv = document.getElementById('message');

// 1. Обработка focus на email
emailInput.addEventListener('focus', () => {
emailHint.textContent = 'Введите корректный email';
});

// 2. Обработка blur на email
emailInput.addEventListener('blur', () => {
emailHint.textContent = '';
});

// 3. Обработка change на password
passwordInput.addEventListener('change', function() {
if (this.value.length < 6) {
    this.style.border = '2px solid red';
} else {
    this.style.border = '';
}
});

// 4. Обработка submit формы
form.addEventListener('submit', function(event) {
event.preventDefault(); // Отменяем стандартную отправку формы

let isValid = true;

// Проверка email
if (!emailInput.value) {
    isValid = false;
    emailInput.style.border = '2px solid red';
} else {
    emailInput.style.border = '';
}

// Проверка пароля
if (passwordInput.value.length < 6) {
    isValid = false;
    passwordInput.style.border = '2px solid red';
} else {
    passwordInput.style.border = '';
}

// Вывод сообщения
if (isValid) {
    messageDiv.textContent = 'Форма успешно отправлена!';
    messageDiv.style.color = 'green';
    // Здесь обычно делается отправка данных на сервер, например, через fetch
} else {
    messageDiv.textContent = 'Исправьте ошибки в форме';
    messageDiv.style.color = 'red';
}
});
