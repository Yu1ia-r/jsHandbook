Объекты

Что такое объекты?

Объекты в JavaScript — это именованные коллекции свойств, где каждое свойство состоит из ключа (имени) и значения. Если функции можно представить как действия, то объекты — это сущности, которые хранят данные и могут иметь поведение через методы.

Пример объекта:

const person = {
  name: "Иван",
  age: 25,
  greet: function() {
console.log("Привет, менязовут " + this.name);
  }
};

console.log(person.name); // "Иван"
person.greet(); // "Привет, меня зовут Иван"

Зачем нужны объекты?

С помощью объектов программы становятся более структурированными. Объекты позволяют:

  1. объединять связанные данные
  2. описывать сущности реального мира
  3. хранить как данные, так и функции для работы с ними
Использование объектов помогает писать чистый и понятный код, где данные логически сгруппированы.

Основы создания объектов

Создать объект в JS можно несколькими способами:

1. Литерал объекта (самый удобный):

constcar = {
brand: "Toyota",
year: 2020
};

2. Черезконструктор Object:

const car = new Object();
car.brand = "Toyota";
car.year = 2020;

3. Через функцию-конструктор:

function Person(name, age) {
  this.name = name;
this.age = age;
}
const user = new Person("Анна", 22);
console.log(user.name); // "Анна"

Свойства и доступ к ним

Свойства — это данные, хранящиеся в объекте. Доступ к ним возможен двумя способами: 

console.log(car.brand);     // "Toyota" (через точку)
console.log(car["year"]);   // 2020 (через строку-ключ)

Методы объектов

Метод — это функция, которая является свойством объекта.

const calculator = {
add(a, b) {
    return a + b;
  },
subtract(a, b) {
    return a - b;
  }
};

console.log(calculator.add(5, 3)); // 8

Вложенные объекты

Объект может содержать другие объекты:

const student = {
  name: "Оля",
  marks: {
    math: 5,
english: 4
  }
};
console.log(student.marks.math); // 5

Перебор свойств

Для перебора всех свойств объекта используют цикл for...in:

const user = { name: "Ира", age: 30 };
for (let key in user) {
console.log(key + ": " + user[key]);
}
// name: Ира
// age: 30

Ключевое слово this

Внутри методов объекта this указывает на сам объект.

const person = {
  name: "Максим",
greet() {
console.log("Привет, я " + this.name);
  }
};
person.greet(); // "Привет, я Максим"

Создание копий объектов

Важно помнить, что при присвоении копируется ссылка на объект, а не сам объект.

const obj1 = { a: 1 };
const obj2 = obj1;
obj2.a = 5;
console.log(obj1.a); // 5

Чтобы создать независимую копию, используют:

const copy1 = Object.assign({}, obj1);
const copy2 = { ...obj1 };

Примеры полезных объектов

1. Счётчик с методами:

constcounter = {
value: 0,
increment() {
this.value++;
  },
decrement() {
this.value--;
  }
};
counter.increment();
console.log(counter.value); // 1

2. Фабрика для создания пользователей:

functioncreateUser(name, age) {
return {
name,
age,
greet() {
console.log(`Привет, я ${this.name}, мне ${this.age} лет.`);
    }
  };
}
const user1 = createUser("Анна", 22);
user1.greet(); // "Привет, я Анна, мне 22 лет."

3. Хранение настроек:

constsettings = {
theme: "dark",
language: "ru",

Интересные и полезные примеры с использованием объектов

1. Корзина товаров:

constcart = {
items: [],
addItem(item) {
this.items.push(item);
  },
showItems() {
    console.log(this.items);
}
};
cart.addItem("Яблоко");
cart.addItem("Банан");
cart.showItems(); // ["Яблоко", "Банан"]

2. Калькулятор с объектом операций:

constcalculator = {
operations: {
    "+": (a, b) => a + b,
    "-": (a, b) => a - b,
    "*": (a, b) => a * b
  },
calculate(a, b, op) {
    return this.operations[op](a, b);
}
};
console.log(calculator.calculate(5, 3, "+")); // 8

3. Счётчик с замыканием через объект:

function createCounter() {
  let value = 0;
  return {
increment() {
      value++;
      return value;
    },
decrement() {
      value--;
      return value;
    }
  };
}
const counter1 = createCounter();
console.log(counter1.increment()); // 1
console.log(counter1.increment()); // 2

Задания для самостоятельной работы:

1. Создайте объект book, который хранит название, автора и год издания. Добавьте метод info(), который выводит строку: Книга <название>, автор <автор>, издана в <год>. 

const book = {
  title: "Война и мир",
  author: "Лев Толстой",
  year: 1869,

  // Метод для вывода информации о книге
  info: function() {
    console.log(`Книга "${this.title}", автор ${this.author}, издана в ${this.year} году.`);
  }
};

// Проверка работы метода
book.info(); // Выведет: Книга "Война и мир", автор Лев Толстой, издана в 1869 году.

2. Создайте объект bankAccount с полем balance и методами deposit(amount) и withdraw(amount).

const bankAccount = {
  balance: 0, // Начальный баланс

  // Метод для пополнения счета
  deposit: function(amount) {
    if (amount > 0) {
      this.balance += amount;
      console.log(`Счет пополнен на ${amount}. Текущий баланс: ${this.balance}.`);
    } else {
      console.log("Сумма пополнения должна быть положительной.");
    }
  },

  // Метод для снятия средств
  withdraw: function(amount) {
    if (amount > this.balance) {
      console.log(`Недостаточно средств. Ваш баланс: ${this.balance}.`);
    } else if (amount <= 0) {
      console.log("Сумма для снятия должна быть положительной.");
    } else {
      this.balance -= amount;
      console.log(`Вы сняли ${amount}. Текущий баланс: ${this.balance}.`);
    }
  }
};

// Проверка работы
console.log(`Начальный баланс: ${bankAccount.balance}`);
bankAccount.deposit(1000); // Пополняем счет
bankAccount.withdraw(300); // Снимаем средства
bankAccount.withdraw(800); // Пытаемся снять больше, чем есть на счете
console.log(`Итоговый баланс: ${bankAccount.balance}`);

3. Напишите функцию createStudent(name, group), которая возвращает объект со свойствами name, group и методом introduce(). 

function createStudent(name, group) {
  return {
    name: name,
    group: group,

    // Метод для представления студента
    introduce: function() {
      console.log(`Студент ${this.name} из группы ${this.group}.`);
    }
  };
}

// Проверка работы
const student1 = createStudent("Иван Петров", "ИС-21");
const student2 = createStudent("Мария Сидорова", "ЭК-22");

student1.introduce(); // Выведет: Студент Иван Петров из группы ИС-21.
student2.introduce(); // Выведет: Студент Мария Сидорова из группы ЭК-22.
